# Research Portfolio

Static portfolio site for GitHub Pages.

## Files

- `index.html` is the homepage.
- `projects/` contains the individual project case-study pages.
- `styles.css` and `script.js` are shared by all pages.
- `assets/Parsa_Riazi_Bakhshayesh_Resume.pdf` is linked from the site.

## Deploy

1. Put these files at the root of your `username.github.io` repository.
2. Commit and push to the default branch used by GitHub Pages.
3. If needed, enable Pages in the repository settings and publish from the repository root.

No build step is required.
