SUNLAB · Jan 2025–May 2025

Generating medical images from richer patient state, not isolated labels.

This project combines structured patient context with image synthesis. The main idea is that synthetic imaging becomes more useful when the generative model is conditioned on a representation of the visit, while latent patient dependencies are modeled explicitly instead of treated as flat metadata.

Conditioning signal Multi-modal visit representation rather than single-field prompting.

System design

Three layers working together

  • Encode multi-modal patient visits so the model sees more than sparse codes alone.
  • Use a Graph Neural Network to model latent dependencies between demographics and sparse medical codes.
  • Condition a diffusion model on that structured latent state to synthesize high-fidelity images.
Target applications The pipeline was designed around Digital Twin and External Control Arm settings where higher-quality synthetic data can reduce scarcity and improve downstream experimentation.

What makes it interesting

  • Bridges structured clinical data and visual generation.
  • Uses graph structure to avoid flattening sparse patient context.
  • Focuses on controllability, not just image realism.

Stack

Diffusion GNN Clinical representation learning

1. Build a visit representation

The conditioning signal integrates multiple sources from the patient visit so the generator receives a more coherent latent summary.

2. Model relationships explicitly

A graph layer captures dependencies between demographics and sparse codes that would be easy to lose in a purely tabular embedding.

3. Synthesize with intent

The diffusion pipeline is framed as a controlled synthetic data system, useful for downstream clinical AI rather than an image generator in isolation.
